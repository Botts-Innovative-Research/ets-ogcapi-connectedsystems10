OGC API - Connected Systems Conformance Test Suite

This executable test suite verifies implemented portions of OGC API - Connected Systems Part 1 (OGC 23-001) and implemented portions of Part 2 (OGC 23-002). It exercises externally visible HTTP behavior of the implementation under test.

TeamEngine 6 is the forward runtime. Local OSH is the primary development E2E target; GeoRobotix is advisory only.

The test run arguments are summarized in the following table. The Obligation descriptor can have the following values: M (mandatory) or O (optional).

Test run arguments
Name Value domain Obligation Description
iut URI M CS API landing page URL for the implementation under test.
auth-credential String O Opaque Authorization header value used for authenticated local runs.
mutation-tests-enabled Boolean O Set to true only when running against a dedicated mutable IUT.
mutation-iut-policy String O Policy acknowledgement required by mutation tests.
